Ce dossier contient les routes Express (API v1) :

- `auth.js` : register/login/refresh/logout/me
- `profiles.js` : profils candidate/recruiter
- `jobs.js` : offres (publiques + recruteur + admin approve)
- `cvs.js` : CV + sections JSON (contrôle accès recruteur via candidature)
- `applications.js` : candidatures + progression de statut
- `admin.js` : stats + users
